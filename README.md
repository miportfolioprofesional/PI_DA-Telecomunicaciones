## KPIs (Indicadores Calve de Desempe√±o)

`KPI Tasa de crecimiento Accesos por cada 100 hogares (KPI_1)`

> Este indicador mide el cambio porcentual en la cantidad de accesos a internet por cada 100 hogares en un per√≠odo de tiempo determinado. La f√≥rmula para calcularlo es: KPI_1 = ((Nuevo acceso - Acceso actual) / Acceso actual) * 100. `KPI Tasa de crecimiento Mbps (Media de bajada) (KPI_2)`

> Este indicador eval√∫a el cambio porcentual en la velocidad media de descarga de internet en un per√≠odo espec√≠fico. La f√≥rmula es: KPI_2 = ((Media nueva - Media actual) / Media actual) * 100. `KPI Porcentaje de cobertura de acceso a internet por habitantes`

> Este indicador muestra el porcentaje de la poblaci√≥n que tiene acceso a internet en un √°rea determinada, generalmente expresado como un porcentaje. La f√≥rmula para calcularlo es: % Cobertura = (‚ÄòTotal‚Äô * 100) / ‚ÄòPoblaci√≥n‚Äô. Simplemente multiplica la cantidad total de acceso a internet por 100 y luego divide el resultado por la poblaci√≥n total para obtener el porcentaje de cobertura.
